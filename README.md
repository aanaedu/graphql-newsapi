# graphql-newsapi
A sample express graphql implementation for NewsAPI (https://newsapi.org)

# Quick Start Guide

### Installation & Startup

Assuming you have node and nodemon installed, simply enter the below in the terminal window:

```
$ git clone https://github.com/aanaedu/graphql-newsapi.git
```
To install the dependencies, enter the following in your terminal:
```
$ cd graphql-newsapi
$ npm install
```

### Local Environment Variables
Create a file named .env in the root directory. This file should contain:
```
API_KEY=your-api-key-here
```

### Starting the App
To start the app, make sure you are in the project directory and enter the following: 
```
$ npm start
```
*This requires nodemon to be installed globally*
